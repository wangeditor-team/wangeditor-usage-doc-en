# Editor Attrs
You can get some editor info by attrs as follows.

## editor.id
The unique id of editor.

## editor.config
The all configs of editor. 

## editor.$textElem.elems[0]
The dom of editable area.

## editor.textElemId
The id of editor text element.

## editor.$toolbarElem.elems[0]
The dom of editor menu.

## editor.toolbarElemId
The id of editor menu element.
